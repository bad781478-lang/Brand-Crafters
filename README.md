# Brand Crafters

Production-ready Vite/React site for Brand Crafters.

## Deploy

- Build command: `npm run build`
- Publish directory: `dist`
- Preview locally: `npm run preview`

The project includes SPA fallback config for Netlify (`public/_redirects` and `netlify.toml`) and Vercel (`vercel.json`) so direct links such as `/services/brand-identity` load correctly after deployment.
